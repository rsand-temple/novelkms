# novelkms
Novel authoring knowledge management system

To run

```
C:\Users\rsand\git\novelkms\novelkms-distro\target>java -jar novelkms.jar server ..\..\novelkms-backend\src\main\conf\config.yaml
```

To build

```
cd C:\Users\rsand\git\novelkms
mvn package
```

To run in Eclipse

```
Run as... Java Application
Class: com.richardsand.novelkms.NovelKmsServer
Program arguments: server ..\..\novelkms-backend\src\main\conf\config.yaml
```

To run Vite front-end

```
cd c:\users\rsand\git\novelkms\novelkms-frontend
npm run dev
```
