# The Codex

The **Codex** is your project knowledge base: characters, canon, world details, timeline notes, and anything else worth keeping beside the manuscript. Each book's Codex lives at the bottom of its branch in the [navigation tree](#help:manuscript.nav-tree).

- Codex **types** group related entries and define the fields those entries fill in.
- Codex **entries** hold the actual notes and edit like scenes.

A new Codex starts with seven types — Characters, Voices, Plot, World, Timeline, Canon, and Notes — but they are only a starting point. You can rename them, change their fields, and add your own types; see [Codex Types and Fields](#help:codex.types).

Entries can be created directly, or [promoted from AI review findings](#help:ai.promotion). Like manuscript items, deleted Codex entries and types go to [Trash](#help:manuscript.trash) and can be restored.
