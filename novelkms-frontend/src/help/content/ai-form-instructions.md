# Form Instructions

The AI review prompt has two parts. The **functional** part — the output format NovelKMS needs — is fixed. The **form** part — the editorial persona and behavioral constraints — is yours to edit.

Form instructions resolve with single-block selection (no blending) across:

**Book → Project → User global → System default**

The system default is built in and read-only; your global is editable; project and book are optional standalone overrides. Each review permanently records the exact form text and which scope it came from, so old reviews stay faithful even after you change your global.

For a one-off nudge rather than a persistent persona, use [one-time guidance](#help:ai.guidance).
