# The Context Menu

Right-click any node in the [navigation tree](#help:manuscript.nav-tree) for actions relevant to it. The available items depend on the node type.

## Manuscript nodes (chapter, scene, book, part, project)

- **Add** a child node (book, part, chapter, scene, or Codex entry).
- **Rename** inline (or press F2 on a selected node).
- **Delete** — moves the item to [Trash](#help:manuscript.trash).
- **Reset chapter numbering** from a chosen chapter.

## Chapter-specific AI actions

- **Generate / Edit… / Clear** memory document — creates or manages the chapter's [memory document](#help:ai.memory).
- **Generate / Edit… / Clear** chapter summary — creates or manages the chapter's [summary](#help:ai.summaries).
- **Generate / Edit… / Clear** editorial — creates or manages the chapter's [editorial](#help:ai.editorial).

## Book-specific AI actions

- **View chapter summaries…** — shows all chapter summaries and the book summary.
- **Generate / Edit… / Clear** book summary.

## Artifact nodes (folder, file)

- **Rename** — rename the file or folder in place.
- **Move to…** — move into a different folder.
- **Move to Trash** — soft-deletes; recoverable from Trash.
- **Download** (files only).
- **Preview** (image files only) — opens a read-only preview modal.

See also: [Artifacts](#help:manuscript.artifacts), [Trash](#help:manuscript.trash).
