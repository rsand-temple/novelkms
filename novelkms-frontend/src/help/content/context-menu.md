# The Context Menu

Right-click any node in the [navigation tree](#help:manuscript.nav-tree) for actions relevant to it:

- **Add** a child (book, part, chapter, scene, or Codex entry).
- **Rename** inline (or press F2).
- **Delete** — moves the item to [Trash](#help:manuscript.trash).
- **AI actions** on chapters and books: generate a [memory document](#help:ai.memory) or a [summary](#help:ai.summaries).
- **Reset chapter numbering** from a chosen chapter.

The available actions depend on the node type.
