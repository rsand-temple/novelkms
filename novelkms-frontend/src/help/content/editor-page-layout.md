# Page Layout

Page layout controls paper size, margins, and whether layout preview is enabled. It affects **export and preview only** — never the live editor.

Layout resolves across **Book → Project → System** scopes, set from the settings dialog with a per-tab override toggle (off inherits; on creates an editable copy).

This is a drafting aid, not a full typesetting engine.
