# Cover & Part Templates

Templates define the **cover page** and **part pages** of a book, using field tokens that fill in automatically — title, author, word count, part number, and the like.

- Edit **global** templates from the Templates menu in the top bar.
- Override a template **per book** from Book Properties; an override is a copy you can edit freely. Deleting it reverts to the global.

Insert tokens from the editor's *Insert field* menu and toggle preview to see them filled in. Token values are substituted only in preview — your stored template keeps the tokens.
