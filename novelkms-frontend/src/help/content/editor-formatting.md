# Formatting & Structure

The editor toolbar offers the building blocks of manuscript text:

- **Inline:** bold, italic, underline, strikethrough.
- **Blocks:** headings, bullet and numbered lists, block quotes.
- **Scene breaks:** a horizontal rule that also marks where one scene ends and the next begins in aggregate editing.
- **Images:** inserted inline and stored with the scene.

For consistent look-and-feel across the manuscript, prefer [paragraph styles](#help:editor.styles) over manual formatting where you can.
