# Importing

Use the **Import** menu in the top bar:

- **From Word (.docx)** — parses a Word document into a new book, detecting cover pages, parts, chapters, scene breaks, and inline images.
- **From NovelKMS archive (.json)** — restores a previously exported project archive.

Markdown import is planned. After a Word import, review the detected structure in the [navigation tree](#help:manuscript.nav-tree) and adjust with [drag and drop](#help:manuscript.drag-drop) as needed.
