# The Editor

The center panel is a rich-text editor built for manuscript prose. Whatever you select in the [navigation tree](#help:manuscript.nav-tree) determines what it shows:

- A **scene** edits that one scene.
- A **chapter**, **part**, or **book** edits all the scenes beneath it at once, while still saving each scene independently behind the scenes.

Your work autosaves as you type. Use the toolbar for [formatting](#help:editor.formatting), [paragraph styles](#help:editor.styles), and [search & replace](#help:editor.search).

Related: [templates](#help:editor.templates), [page layout](#help:editor.page-layout).
